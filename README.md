# Brain-Vascular-Dataset-Analysis
This code is from a hackathon run by Binfsoc of UNSW and COMBINE, processing raw single-cell expression data.
The output is visual representation of the cell expression data for visual analysis, as well as statistical gene processing, indicating prominently up or down regulated genes

Data set used was from Brain vascular single-cell multi-omics elucidates disease risk associations (Reid et al., 2025).
